例题一：P1073 最优贸易

考虑到没买->买了->卖了的严格顺序，我们想出了以下做法：

做法一：

把图分三层：

  • 还没买水晶球

  • 买了水晶球还没卖

  • 卖了水晶球

然后加上“买水晶球”（dis[i][1]<-dis[i][0]-val[i]），“卖水晶球”（dis[i][2]<-dis[i][1]+val[i]）两种特殊边做最短路。

显然你已经意识到了买、卖之间时间顺序的重要。

那么再仔细想想：是不是可以买从前面找最好的买的地方，从后面反过来找最好的卖的地方？

做法二：

从起点开始刷最短路，求沿途最低的val[i]。

从终点开始反刷最短路，求沿途最高的val[i]。

ans=max(dis2[i]-dis1[i])。

然而需要反向建图，常数和做法一差不多……

例题二：P4083 [USACO17DEC]A Pie for a Pie

无毒翻译（感谢ZS）：

看似是SB题，连边倒跑最短路即可。

然而！动手写的时候，你会发现一个重大问题：存边！

显然老老实实存边是不可能的，这辈子都不可能的。所以我们考虑用到一个节点的时候再提取对应边。

为了让提取对应边更快，我们复制Bessie馅饼数组的拷贝和Elsie馅饼数组的拷贝并排序。有序就意味着对方的可以连边的馅饼聚在一块，那么我们二分找到第一个馅饼就一下子把一串馅饼都提出来了。

不难发现，还是有几乎每对馅饼都能连边的极端情况。要是有这种极端情况此题就unzuoable了。幸好没有。

int inline read(){
    int num=0,neg=1;char c=getchar();

    while(isdigit(c)) num=num*10+c-'0',c=getchar();
    return num*neg;
}

P5304 [GXOI/GZOI2019]旅行者

题意及其简单。就是求K个点间两两距离的最小值。

然而直接交一个Floyd只有10分……

假设求一个点集合到另一个点集合的距离的最小值，我们可以直接把第一个点集中所有点的dis都初始化为0，然后跑最短路，直接调第二个点集中所有点dis的最小值。同样还要以第二个点集为起点再跑一次。这个过程（如果用dijkstra实现最短路）是O(Nlog⁡N)的。

也许，我们可以找到一种把整个点集不断划分成两个集合的方案，使得任何两个点总在某次分割中不在同一个集合中，而这样的方案具有较少的分割次数。

注意到（我也不知道是怎么注意到的……），任何不相等的两个数，它们的二进制总是至少有一位不同，所以我们第i次分割时按每个数二进制第i位是0还是1进行分割：第一个集合中的点编号二进制第i位都是0，另一个都是1。这样的分割次数显然是log⁡N的，于是我们得到一个Nlog2⁡N的算法。

代码如下（常数巨大，需要O2，不过卡常空间还是很大的）：

// luogu-judger-enable-o2
#include <bits/stdc++.h>
#define ll long long
using namespace std;


int T,N,M,K; 
int A[100005];
int lnk[100005];
int pre[500005],tgt[500005],val[500005],cnt;
void add_E(int u,int v,int c){
    pre[++cnt]=lnk[u],tgt[cnt]=v,val[cnt]=c,lnk[u]=cnt;
}


struct node{
    int pos;ll val;
    bool operator <(const node b)const{return val>b.val;}
}H[500005];int len;
void putin(node x){H[++len]=x;push_heap(H+1,H+len+1);}
node putout(){pop_heap(H+1,H+len+1);return H[len--];}


ll dis[500005];bool vis[500005],is_tgt[500005];
ll ans;
void Dij(){
    while(len){
        node u=putout();
        while(vis[u.pos]&&len) u=putout();
        if(vis[u.pos]) return;
        vis[u.pos]=1;
        for(int e=lnk[u.pos];e;e=pre[e]){
            node v;v.pos=tgt[e];
            if(dis[u.pos]+val[e]<dis[v.pos]){
                v.val=dis[v.pos]=dis[u.pos]+val[e];
                putin(v);
            }
        }
    }
}


int main(){
    scanf("%d",&T);
    while(T--){
        scanf("%d%d%d",&N,&M,&K);
        memset(lnk,0,sizeof(lnk));cnt=0;
        for(int i=1;i<=M;i++){
            int u,v,c;scanf("%d%d%d",&u,&v,&c);
            add_E(u,v,c);
        }
        for(int i=1;i<=K;i++) scanf("%d",&A[i]);
        ans=0x3f3f3f3f3f3f3f3f;
        for(int i=0;(1<<i)<=N;i++){
            memset(dis,63,sizeof(dis));
            memset(vis,0,sizeof(vis));
            memset(is_tgt,0,sizeof(is_tgt));
            for(int j=1;j<=K;j++)
                if(A[j]&(1<<i)) dis[A[j]]=0,putin((node){A[j],0});
            Dij();
            for(int j=1;j<=K;j++)
                if(!(A[j]&(1<<i))) ans=min(ans,dis[A[j]]);
            
            memset(dis,63,sizeof(dis));
            memset(vis,0,sizeof(vis));
            memset(is_tgt,0,sizeof(is_tgt));
            for(int j=1;j<=K;j++)
                if(!(A[j]&(1<<i))) dis[A[j]]=0,putin((node){A[j],0});
            Dij();
            for(int j=1;j<=K;j++)
                if(A[j]&(1<<i)) ans=min(ans,dis[A[j]]);
        }
        printf("%lld\n",ans);
    }
    
    return 0;
}
