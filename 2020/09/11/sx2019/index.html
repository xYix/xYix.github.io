йўҳзӣ®й“ҫжҺҘ

еҜ№йә»е°Ҷзҡ„зғӯзҲұж¶ҲеӨұж®Ҷе°ҪдәҶпјҲжӮІпјү

иҖғиҷ‘жҖҺд№ҲеҲӨж–ӯдёҖеүҜзүҢжҳҜиғЎзҡ„гҖӮжҲ‘д»¬жңүдёҖдёӘ DP еҒҡжі•пјҡи®ҫ $f_{i,j_1,j_0,k}$ жҳҜиҖғиҷ‘йқўеҖјдёә $1вҖҰi$ зҡ„жүҖжңүзүҢпјҢйў„з•ҷзҡ„ $(i-1,i,i+1)$ йЎәеӯҗж•°зӣ®дёә $j_1$пјҢйў„з•ҷзҡ„ $(i,i+1,i+2)$ йЎәеӯҗж•°зӣ®дёә $j_0$пјҢ$k$ жҳҜжңүжІЎжңүйў„з•ҷиҝҮеҜ№еӯҗпјҢзҡ„жңҖеӨ§йқўеӯҗж•°гҖӮиҝҳиҰҒеҶҚеҠ дёҖз»ҙиЎЁзӨәжҖ»е…ұжңүеӨҡе°‘еҜ№еӯҗжқҘеҲӨж–ӯдёғеҜ№еӯҗгҖӮжҳҫз„¶ $j_1,j_0<3$пјҢеҗҰеҲҷеҸҜд»ҘжӢҶжҲҗеҲ»еӯҗгҖӮ

еҲӨж–ӯиғҪеҗҰиғЎзүҢеҫҲз®ҖеҚ•пјҢ$f_i,\star,\star,1$ дёӯжңүдёҖдёӘеӨ§дәҺзӯүдәҺ $4$ жҲ–иҖ…жңүдёғдёӘеҜ№еӯҗеҚіеҸҜгҖӮ

жҳҫз„¶иҝҷдёӘ DP е’Ң $i$ е®Ңе…ЁжІЎжңүе…ізі»пјҢе’Ң $k$ д№ҹе…ізі»дёҚеӨ§пјҢе®ғеҸҜд»ҘзңӢжҲҗдёҖдёӘ $2\times 3\times 3$ зҡ„зҹ©йҳөпјҢжҺҘеҸ—дёҖдёӘж•°еҗҺиҪ¬з§»жҲҗеҸҰдёҖдёӘзҹ©йҳөпјҢеҸҜд»ҘзңӢжҲҗдёҖдёӘиҮӘеҠЁжңәгҖӮиө°еҲ°иғЎиҠӮзӮ№еҚіиғЎзүҢгҖӮ

йӮЈд№ҲжҲ‘д»¬е°ұеҸҜд»ҘиҖғиҷ‘еңЁиҝҷдёӘиҮӘеҠЁжңәдёҠ DP дәҶпјҢи®ҫ $g_{i,j,p}$ жҳҜиҖғиҷ‘дәҶйқўеҖјдёә $1вҖҰi$ зҡ„зүҢпјҢж‘ёдәҶе…¶дёӯ $j$ еј пјҢиө°еҲ°дәҶиҮӘеҠЁжңәиҠӮзӮ№ $p$ зҡ„жҰӮзҺҮгҖӮжҲ‘д»¬йғҪзҹҘйҒ“иғЎзүҢжңҹжңӣе°ұжҳҜ $\sum_{i=0}^\infty P(\texttt{ж‘ёдәҶ i еј зүҢиҝҳдёҚиғЎ})$пјҢеӨ§еҠӣ DP еҚіеҸҜгҖӮ

д»Јз ҒеҰӮдёӢпјҢеҶҷеҫ—з®Җзӣҙи®©дәәжғіеҗҗпјҡ

T3

йўҳзӣ®й“ҫжҺҘ

иҖғиҷ‘е·®еҲҶпјҡжұӮзЁіе®ҡеәҰ $\le k$ зҡ„йӣҶеҗҲж•°йҮҸгҖӮ

и®°жҹҗдёӘзӮ№ $u$ зҡ„жқғеҖјдёә $v(u)$гҖӮиҖғиҷ‘ $v=W$ зҡ„иҠӮзӮ№жҳҫз„¶еҪўжҲҗдәҶдёҖжқЎз”ұж №еҲ°еҸ¶еӯҗзҡ„й“ҫгҖӮжҜҸдёӘе…¶д»–еҸ¶еӯҗд№ҹдјҡеҗ‘дёҠе»¶дјёеҮәдёҖжқЎй“ҫпјҢзӣҙеҲ°ж’һеҲ°дё»й“ҫжҲ–иҖ…е…¶д»–й“ҫгҖӮ

еҸӘиҰҒдё»й“ҫдёҠжңүдёҖдёӘзӮ№ж”№еҸҳ $W$ е°ұдј дёҚеҲ°ж №дәҶпјӣдәҺжҳҜеҸҜд»Ҙж–ӯејҖдё»й“ҫдёҠзҡ„жүҖжңүиҫ№еҲҶеҲ«иҖғиҷ‘гҖӮ

иҖғиҷ‘еҜ№еҸ¶еӯҗиҠӮзӮ№зҡ„зӯ–з•ҘжҳҜд»Җд№ҲпјҹеҰӮжһңеҪ“еүҚж №зҡ„ж·ұеәҰдёәеҘҮж•°пјҢйӮЈд№Ҳеҝ…з„¶еҸ– $W+1$пјҡж—ўиғҪжңҖз»ҲиғңиҝҮ $W$пјҢеҸҲжӣҙдёҚе®№жҳ“еӣ дёәеӨӘеӨ§еңЁеҒ¶ж•°ж·ұеәҰиў«ж·ҳжұ°гҖӮеҒ¶ж•°еҗҢзҗҶгҖӮдәҺжҳҜеҜ№дәҺзЎ®е®ҡзҡ„ $k$пјҢжҜҸдёӘеҸ¶еӯҗзҡ„зӯ–з•ҘжҳҜзЎ®е®ҡзҡ„пјҢе’Ң rand() еҮәзҡ„йӣҶеҗҲж— е…ігҖӮ

иҖғиҷ‘ DPпјҡ$\text{f}_u$ иЎЁзӨәпјҡеҜ№дәҺж·ұеәҰдёәеҘҮж•°зҡ„ $u$пјҢжңүеӨҡе°‘з§ҚеҸ¶еӯҗйӣҶеҗҲиғҪи®© $v(u)>W$гҖӮеҜ№дәҺеҒ¶ж•°еҸҚд№ӢгҖӮ

вҖңиғҪвҖқеӯ—еңЁиҝҷйҮҢеҸҜиғҪеј•еҸ‘иҜҜи§ЈпјҡеҜ№дәҺжҜҸдёӘйӣҶеҗҲпјҢзӯ–з•ҘжҳҜе®Ңе…ЁзЎ®е®ҡзҡ„пјҢдёҚдјҡжңүе“ӘдёӘйӣҶеҗҲж—ўвҖңиғҪвҖқдҪҝ $v\le W$ еҸҲвҖңиғҪвҖқдҪҝ $v>W$гҖӮ

и®° $u$ еӯҗж ‘дёӯзҡ„еҸ¶еӯҗж•°дёә $\text{siz}_u$гҖӮ

жҳҺзЎ®е®ҢдәҶдёҠйқўеҮ зӮ№пјҢжҺҘдёӢжқҘеҶҷ DP ж–№зЁӢгҖӮ

еҜ№дәҺеҸ¶еӯҗиҠӮзӮ№пјҢ

жңҖз»Ҳи®Ўз®—зӯ”жЎҲж—¶еҸӘйңҖиҰҒжҠҠжҜҸдёӘдё»й“ҫдёҠзҡ„иҠӮзӮ№д№ҳиө·жқҘгҖӮ

еҪ“ $k$ еҸҳеҠЁж—¶жҜҸдёӘеҸ¶еӯҗзҡ„ $\text{f}$ еҸӘдјҡеҸҳеҠЁдёҖж¬ЎпјҢжҸҗзӨәжҲ‘д»¬еҠЁжҖҒ DPгҖӮдәҺжҳҜе°ұеҒҡе®ҢдәҶгҖӮжіЁж„Ҹз”ұдәҺ $\text{f}_v$ еҸҜиғҪдёә 0пјҢиҖҢйҷӨд»Ҙ 0 йқһеёёйә»зғҰпјҢжүҖд»ҘиҖғиҷ‘д»ҺеӨ§еҲ°е°Ҹжһҡдёҫ $k$гҖӮ

жҖқи·ҜйқһеёёзәҝжҖ§пјҢеҶ·йқҷеҲҶжһҗдёҖдёӢеә”иҜҘдёҚз®—еӨӘйҡҫпјҹйӮЈд№Ҳй—®йўҳжқҘдәҶиҜҘеҰӮдҪ•еӯҰдјҡеҶ·йқҷеҲҶжһҗе‘ў
